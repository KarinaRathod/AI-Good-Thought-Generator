
# 🧠✨ AI Good Thought Generator

An AI-powered application that generates **personalized, meaningful positive thoughts** based on your mood and life situation using Google Gemini.

---

## 🚀 Features

- 💭 Mood-based thought generation (Happy, Sad, Anxious, etc.)
- 📝 Situation-aware responses (personalized insights)
- 🎭 Tone control (Friendly, Motivational, Calm, Strict)
- 📜 Thought history tracking
- 📊 Mood trend visualization
- 📥 Downloadable thought journal
- ⚡ Clean and interactive UI with Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- Matplotlib
- Session State (for memory)

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-good-thought-generator.git
cd ai-good-thought-generator
````

---

### 2️⃣ Install Dependencies

```bash id="install-good-thought"
pip install -r requirements.txt
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file:

```env id="env-good-thought"
GOOGLE_API_KEY=your_gemini_api_key
```

Get your API key:
[https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)

---

### 4️⃣ Run the App

```bash id="run-good-thought"
streamlit run app.py
```

---

## 🧪 How It Works

1. Select your **mood**
2. Choose a **tone**
3. Describe your **current situation**
4. Click **Generate Thought**
5. Get a **personalized, meaningful insight**
6. Track your thoughts and mood over time
